## 快速开始

目前默认的AI提供商是OpenAI，您将需要从OpenAI生成API密钥

 运行 `k8sgpt auth new` 在k8sgpt中设置。

 运行 `k8sgpt filters` 以管理分析仪使用的活动过滤器。默认情况下，在分析期间执行所有过滤器。

+ 运行 `k8sgpt analyze` 以运行扫描。
+ 并使用 `k8sgpt analyze --explain` 获取有关问题的更详细说明。
